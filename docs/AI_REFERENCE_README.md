# AI Reference README

Use these local reference docs when working on Unity AI, Codex MCP, or AI-assisted game-design tasks:

- `docs/UNITY_AI_CODEX_CLI_SETUP.md`
- `docs/UNITY_AI_KNOWLEDGE_INDEX.md`
- `docs/AI_ASSISTED_GAME_DESIGN_WORKFLOW.md`

These files are an index and operating guide. They deliberately avoid copying large Unity/OpenAI documentation into the repository. The installed Unity AI package remains the local source for Unity's full documentation, and official documentation URLs remain the canonical current sources.
