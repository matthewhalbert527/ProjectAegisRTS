# Unity AI Building Source Textures

Promote or copy Unity AI material/Texture2D outputs here using the names documented in:

`docs/UNITY_AI_BUILDING_TEXTURE_PROMPTS.md`

Examples:

- `worn_green_gray_metal_ai_albedo.png`
- `worn_green_gray_metal_ai_normal.png`
- `worn_green_gray_metal_ai_occlusion.png`

The building slate generator uses these textures when present and falls back to deterministic generated textures when they are absent.
