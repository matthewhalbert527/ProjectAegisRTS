# Project Aegis Map Visual Art Pack v2 Rebuild

Install into `unity/Assets/Rts/MapEditor/ArtPack/ProjectAegis_MapVisualArtPack_v1/`. The folder name intentionally remains v1 because current `AegisMapArtPack.Root` is hard-coded to that path. This is production-proxy art for the visual compiler, not final hand-authored art.

After import, have Codex wire `Materials/semantic_materials.json` into `AegisBiomeVisualTheme` so terrain, roads, rivers, cliffs, rocks, and base pads use texture paths instead of color-only rules.
